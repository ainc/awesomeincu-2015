iOS Advanced Course

View Notes

This course will teach you advanced iOS (iPhone, iPad) development. The course is designed for people with some prior experience in object-oriented programming and the Xcode development environment.

  • UI layout - with and without auto-layout
  • Objective-C concepts (ie. categories, subclasses, protocols, blocks)
  • Notification Center
  • Playing Media
  • Programmatic UI
  • Asynchronous programming model
  • Using the Debugger
  • Creating your own protocols & delegates
  • If time permits we'll cover things like Memory management, Using instruments, and how to develop a universal app for both iPhone/iPod and iPad
  • Sandbox time with 1-on-1 support from our instructional team

  • Introduction to web frameworks including Bootstrap and JQuery
  • Sandbox time with 1-on-1 support from our instructional team

Kids Code Camp

Register
View Notes

Instead of your kids spending their free time playing apps, what if they spent it learning to write them? This camp for middle and high school students will teach the fundamentals of coding. Students will create websites, mobile apps and learn to design video games all while having fun with our energetic team. By the end of the week, each student will have built at least 1 web page, mobile app, and video game using HTML, CSS, JavaScript, Android, and Unity 3D. We keep this camp small to ensure everyone gets very focused attention and learns at their own pace. Prepare to be amazed!

  • Introduction to Computer Science concepts with Code.org
  • Learning to write code with JSBin and Codecademy
  • Web Development with HTML, CSS, and JavaScript
  • Mobile App Development with Android
  • Video Game Development with Unity 3D
  • Sandbox time with 1-on-1 support from our instructional team
  • Final project presentations at the end of the week
